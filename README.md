# Heart-Disease-Prediction-ML
Machine Learning project for early heart disease detection
# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Project Description
This project predicts the risk of heart disease using machine learning techniques. It helps in early detection and supports doctors in decision-making.

## 🎯 Objectives
- Predict heart disease risk
- Provide quick and accurate results
- Reduce human errors
- Improve healthcare system

## 📊 Dataset
The dataset includes:
- Age
- Blood Pressure
- Cholesterol
- Heart Rate
- Blood Sugar
- Other medical attributes

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## ⚙️ Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest
- KNN
- SVM
- Naive Bayes

## 🔄 Working
1. Data Collection
2. Data Cleaning
3. Model Training
4. Testing
5. Prediction

## 📈 Features
- Easy to use
- Fast prediction
- Good accuracy
- Reliable system

## ✅ Advantages
- Early detection
- Saves time
- Improves decision making

## ⚠️ Limitations
- Depends on data quality
- Not 100% accurate
